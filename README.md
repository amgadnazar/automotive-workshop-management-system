# Automotive Workshop Marketing and Service Management System

This system consists of two integrated mobile applications designed to support automotive workshop operations and customer services.

### Customer Application

Allows customers to:

* Register and login
* Browse services
* Book appointments
* Order spare parts
* Submit ratings and complaints
* Receive notifications

### Workshop Administrator Application

Allows administrators to:

* Manage services
* Manage bookings
* Manage spare parts inventory
* Manage promotions
* View customer feedback
* Monitor workshop activities

## System Architecture

Frontend:

* React Native

Backend:

* Node.js

Database:

* Firebase Firestore

Authentication:

* Firebase Authentication

Notifications:

* Firebase Cloud Messaging

## Technologies Used

* React Native
* Node.js
* Firebase Firestore
* Firebase Authentication
* Firebase Cloud Messaging
